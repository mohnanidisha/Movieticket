# Movieticket
# C++ Movie Ticket Booking System

## Description
A console-based C++ project that allows users to book and cancel movie tickets.
Includes admin and user login, seat selection, and booking history.

## Features
- User/Admin login
- Seat booking & cancellation
- Linked list based data handling
- Menu-driven interface

## Technologies
- C++
- VS Code

## How to Run
1. Compile using g++
2. Run the executable
